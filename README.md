# 🎓 EduQuest — Геймифицированная платформа для учителей

**EduQuest** — десктопное приложение на PyQt6 + Google Gemini AI, которое превращает подготовку уроков в RPG-приключение. Платформа генерирует квесты, боссов, арены, проверяет домашки и многое другое — всё через ИИ.

## 📋 Модуль 1: Проверка Домашки

### Что это?
ИИ-ассистент, который расшифровывает рукописные конспекты и сочинения учеников с фотографий и проводит жёсткий фактчекинг по биологии. Фокус — только на научной достоверности, орфография и стиль игнорируются.

### Как пользоваться?
1. **Загрузи фото** — нажми «Загрузить фото» и выбери снимок тетради (JPG, PNG, WebP)
2. **Или вставь текст** — если конспект уже расшифрован, вставь его в текстовое поле
3. **Нажми «Сканировать»** — ИИ проанализирует содержание

### Что получишь на выходе?
- **Распознанный текст** — полная расшифровка рукописи
- **Оценка (X/10)** — снижается только за фактические ошибки
- **Список ошибок** — каждая с цитатой, объяснением и правильным ответом
- **Пропущенные понятия** — что ученик забыл упомянуть
- **Выжимка для учителя** — «Ученик хорошо понял X, но плавает в Y»

### Кому полезно?
- **Учителям биологии** — быстрая проверка стопки тетрадей, особенно когда 30+ работ
- **Репетиторам** — анализ конспектов между занятиями, чтобы понять пробелы ученика
- **Ученикам** — самопроверка перед сдачей: загрузил фото → увидел ошибки → исправил
- **Студентам-медикам** — проверка своих конспектов по анатомии, физиологии, цитологии

## 📝 Модуль 2: Генератор квестов

### Что это?
Превращает любую тему урока в полноценный RPG-квест с игровым сеттингом, тремя этапами-испытаниями и финальным боссом. Вместо скучного «откройте учебник на странице 47» — ученики отправляются в приключение.

### Как пользоваться?
1. **Введи тему** — например, «Фотосинтез», «Клеточное деление», «Эволюция»
2. **Укажи класс** — 5, 7, 9, 11 и т.д. (сложность адаптируется)
3. **Нажми «Создать квест»**

### Что получишь на выходе?
- **Название квеста** — например, «Экспедиция в Хлоропластовые Джунгли»
- **Игровой сеттинг** — описание мира, в котором проходит квест
- **3 этапа** — каждый с названием, описанием активности и длительностью в минутах
- **Финальный босс** — имя, описание, условие победы (связано с темой урока)

### Кому полезно?
- **Учителям** — готовый план урока-игры за 30 секунд. Не нужно часами придумывать сценарий
- **Классным руководителям** — для внеклассных мероприятий и предметных недель
- **Методистам** — шаблоны для геймифицированных уроков, которые можно адаптировать
- **Родителям на домашнем обучении** — превратить скучную тему в игру для ребёнка

## 📊 Модуль 3: Адаптивное ДЗ

### Что это?
Анализирует историю оценок и ошибок конкретного ученика, находит слабые места и генерирует персонализированный набор заданий разной сложности. Каждый ученик получает именно те задания, которые ему нужны.

### Как пользоваться?
1. **Введи имя ученика** — «Маша Иванова»
2. **Введи тему** — «Генетика», «Экология» и т.д.
3. **Опиши историю** — «Путает доминантные и рецессивные признаки, не понимает закон Менделя, последняя контрольная — 4/10»
4. **Нажми «Сгенерировать ДЗ»**

### Что получишь на выходе?
- **Анализ слабых мест** — краткая диагностика проблем ученика
- **3-5 заданий** — от лёгких (easy) до сложных (hard), с подсказками
- **Подсказки** — для каждого задания есть хинт, если ученик застрял

### Кому полезно?
- **Учителям** — вместо одного ДЗ на весь класс, даёшь каждому своё. Дифференциация за минуту
- **Репетиторам** — точечная проработка именно тех тем, где ученик плавает
- **Родителям** — понять, над чем ребёнку стоит поработать дома
- **Самим ученикам** — ввести свои ошибки и получить тренировочные задания

## 🏆 Модуль 4: Арена классов

### Что это?
Генератор раундов для командных интеллектуальных соревнований. Каждый раунд — это задание определённого типа (блиц, загадка, дебаты) с очками и таймером. Идеально для разделения класса на команды и проведения баттла.

### Как пользоваться?
1. **Введи тему** — «Кровеносная система», «Биомы Земли»
2. **Нажми «Сгенерировать раунд»** — получишь один раунд
3. **Повторяй** — генерируй столько раундов, сколько нужно для игры

### Что получишь на выходе?
- **Тип раунда** — блиц-вопрос, загадка, дебаты, практическое задание и т.д.
- **Текст задачи** — конкретный вопрос или задание для команд
- **Очки** — сколько баллов стоит этот раунд
- **Лимит времени** — в секундах, чтобы держать темп

### Кому полезно?
- **Учителям** — быстро организовать соревнование на уроке. Разделил класс → запустил раунды → считаешь очки
- **Организаторам олимпиад** — генерация заданий для школьных турниров
- **Вожатым в лагерях** — интеллектуальные игры для отрядов
- **На предметных неделях** — межклассовые баттлы по биологии

## 🧪 Модуль 5: Босс-файт

### Что это?
Контрольная работа в формате RPG-битвы. Ученики «сражаются» с боссом — каждая правильно решённая задача наносит ему урон. У босса есть HP (количество задач), реплики при ошибках и победная фраза. Три уровня сложности.

### Как пользоваться?
1. **Введи тему** — «Митоз и мейоз»
2. **Выбери сложность** — Easy (3 задачи), Medium (5 задач), Hard (7 задач)
3. **Нажми «Создать босса»**

### Что получишь на выходе?
- **Имя босса** — например, «Лорд Анафаза, Разрушитель Веретён»
- **HP босса** — равно количеству задач
- **Реплики** — что босс говорит при ошибке ученика и при правильном ответе
- **Задачи-атаки** — каждая с текстом и подсказкой
- **Победная фраза** — когда ученики побеждают

### Кому полезно?
- **Учителям** — превратить контрольную из стресса в азарт. Ученики сами просят «ещё раунд»
- **Для слабомотивированных классов** — игровая механика вовлекает даже тех, кто обычно скучает
- **На открытых уроках** — эффектная подача, которая впечатлит комиссию
- **Репетиторам** — мини-тесты в формате игры для удержания внимания ученика

## 🕰 Модуль 6: Портал Времени

### Что это?
Интерактивный чат с исторической личностью. ИИ играет роль выбранного учёного или мыслителя (Дарвин, Мендель, Пастер и др.) — говорит от первого лица, реагирует на анахронизмы, может вступить в интеллектуальную дуэль и подарить артефакт за мудрый вопрос.

### Как пользоваться?
1. **Выбери персонажа** — «Чарльз Дарвин», «Грегор Мендель», «Луи Пастер»
2. **Напиши сообщение** — задай вопрос или брось интеллектуальный вызов
3. **Читай ответ** — персонаж отвечает в своём стиле с проверкой исторической точности

### Что получишь на выходе?
- **Реплика персонажа** — в характере и стиле речи исторической личности
- **Эмоция** — гордость, гнев, любопытство и т.д.
- **Историческая справка** — для учителя: что правда, а что художественный вымысел
- **Дуэль** — если ученик спорит, персонаж аргументированно отвечает
- **Артефакт** — за глубокий вопрос персонаж может подарить тематический предмет

### Кому полезно?
- **Учителям** — живая история науки вместо сухих дат. Ученики запоминают через диалог
- **На уроках истории науки** — «поговорить с Дарвином» о его путешествии на Бигле
- **Для проектной работы** — ученик берёт интервью у учёного и оформляет как проект
- **Для одарённых учеников** — интеллектуальные дуэли развивают критическое мышление

## 📋 Модуль 7: Таверна

### Что это?
Диагностический инструмент, который анализирует отзывы учеников или заметки учителя о классе и выдаёт «RPG-статы» — числовые показатели креативности, логики, скорости и внимания. Плюс рекомендация в стиле RPG.

### Как пользоваться?
1. **Введи наблюдения** — «Класс активный в начале, но к концу урока теряет интерес. Хорошо справляются с практикой, но теорию не любят. Несколько учеников постоянно отвлекаются»
2. **Нажми «Анализировать»**

### Что получишь на выходе?
- **Название состояния** — «Уставшие герои», «Бунтующая гильдия» и т.д.
- **Анализ** — 2-3 предложения о настроении класса
- **Статы (0-100):**
  - ✨ Магия — креативность и вовлечённость
  - ⚔ Сила — логика и понимание хард-скиллов
  - 🏃 Ловкость — скорость выполнения задач
  - 🛡 Выносливость — удержание внимания к концу урока
- **Рекомендация** — совет учителю в RPG-стиле

### Кому полезно?
- **Учителям** — быстрая рефлексия после урока. Записал наблюдения → получил диагностику
- **Классным руководителям** — отслеживание динамики класса в течение четверти
- **Школьным психологам** — дополнительный инструмент для анализа групповой динамики
- **Завучам** — сравнение «статов» разных классов для планирования нагрузки

## 🔬 Модуль 8: Кузница артефактов

### Что это?
Создаёт визуальные пособия по запросу учителя: пошаговую инструкцию для рисования на доске + профессиональный промпт для генерации картинки в Stable Diffusion / Midjourney / DALL-E.

### Как пользоваться?
1. **Опиши, что нужно** — «Схема строения клетки», «Цикл Кребса», «Пищевая цепь леса»
2. **Укажи тему** — «Цитология», «Биохимия», «Экология»
3. **Нажми «Создать артефакт»**

### Что получишь на выходе?
- **Название артефакта** — «Карта Клеточной Цитадели»
- **Инструкция для доски** — пошагово: какие фигуры рисовать, как подписать, на что обратить внимание
- **AI-промпт** — готовый английский промпт для генерации картинки (4K, стиль, освещение)
- **Дидактическая ценность** — почему этот визуал поможет понять тему

### Кому полезно?
- **Учителям** — не умеешь красиво рисовать? Следуй инструкции. Нужна картинка? Скопируй промпт в Midjourney
- **При подготовке презентаций** — AI-сгенерированные иллюстрации для слайдов
- **Для стендов и плакатов** — генерация профессиональных изображений для кабинета
- **Для дистанционного обучения** — визуальные материалы для онлайн-уроков

## 🧬 Модуль 9: Бестиарий (Bio-Edition)

### Что это?
Генератор RPG-карточек для биологических микро-объектов. Митохондрия становится «Энергетическим Кристаллом», ВИЧ — «Босс-Вирусом», рибосома — «Кузницей Белка». 100% научная точность + геймерская подача.

### Как пользоваться?
1. **Введи био-объект** — «Митохондрия», «ВИЧ», «Рибосома», «E.coli», «Хлоропласт»
2. **Нажми «Создать карточку»**

### Что получишь на выходе?
- **Название** — на русском и латыни
- **Классификация** — Прокариоты / Эукариоты / Вирусы
- **Научное описание** — 1-2 предложения
- **RPG-карточка:**
  - ❤ HP (0-100) — уровень жизнеспособности
  - ⚔ Тип — игровой класс
  - ✨ Способности (2-3) — реальная биофункция + RPG-эффект
  - 💀 Уязвимости — что может уничтожить/ингибировать
- **AI-промпт** — для генерации иллюстрации в стиле «Dark Fantasy Bio-Punk»

### Кому полезно?
- **Учителям биологии** — распечатай карточки → ученики коллекционируют и учат. Геймификация запоминания
- **Для настольной игры в классе** — «Собери колоду органоидов и сражайся с вирусами»
- **На уроках цитологии** — вместо зубрёжки — живые персонажи с характером
- **Для олимпиадников** — углублённые карточки с латынью и биохимическими деталями
- **Для научпоп-проектов** — готовый контент для школьных журналов и стенгазет

## 🌍 Модуль 10: Ход Игры (Гейм-мастер)

### Что это?
ИИ-гейм-мастер для экологической RPG-симуляции. Класс управляет виртуальной экосистемой: решает, каких животных заселить, какие ресурсы добывать, как реагировать на катастрофы. ИИ рассчитывает последствия и подбрасывает случайные события.

### Как пользоваться?
1. **Опиши действия учеников** — «Команда 1 заселила 10 зайцев, Команда 2 добавила 3 волков, Команда 3 вырубила часть леса»
2. **Опиши текущее состояние** — «Лесная экосистема, баланс 50/100, есть река и поляна»
3. **Нажми «Следующий ход»**

### Что получишь на выходе?
- **Новый балл баланса (0-100)** — 0 = экосистема мертва, 50 = идеал, 100 = перенаселение
- **Последствия** — что произошло из-за действий учеников (научно обоснованно)
- **Случайное событие** — Засуха, Нашествие саранчи, Мутация, Миграция и т.д.
- **Цель на следующий ход** — что нужно сделать, чтобы спасти экосистему

### Кому полезно?
- **Учителям экологии** — многодневная симуляция: каждый урок = один ход. Класс видит последствия своих решений
- **Для проектной деятельности** — группы учеников ведут свои экосистемы и сравнивают результаты
- **На уроках географии** — моделирование биомов и климатических изменений
- **Для экологических кружков** — долгосрочная игра на весь семестр
- **На выездных занятиях** — «что будет, если в этом лесу исчезнут волки?»

## 🛠 Технологии

- **Python 3.11+**

- **Pydantic** — валидация структурированных ответов ИИ
- **LangChain** — оркестрация промптов и structured output

## 🚀 Установка и запуск

### 1. Клонируй репозиторий

```bash
git clone https://github.com/your-username/EduQuest.git
cd EduQuest
```

### 2. Установи зависимости

```bash
pip install PyQt6 langchain-google-genai python-dotenv pydantic
```

### 3. Настрой API-ключ

Создай файл `.env` в папке `EduQuest/`:

```env
GOOGLE_API_KEY=твой_ключ_от_google_ai_studio
```

Получить ключ бесплатно: [Google AI Studio](https://aistudio.google.com/apikey)

### 4. Запусти

```bash
cd EduQuest
python main.py
```

## 🎨 Темы оформления

Приложение поддерживает две темы с переключением в один клик (кнопка в левом нижнем углу сайдбара):

- **🌙 Тёмная** — Dark RPG стиль с золотыми акцентами (по умолчанию)
- **☀ Светлая** — Чистая, тёплая тема с янтарными акцентами

## 🎯 Для кого EduQuest?

| Аудитория | Как использовать |
|-----------|-----------------|
| **Учителя биологии** | Все 10 модулей — от подготовки урока до проверки тетрадей |
| **Учителя других предметов** | Квесты, Арена, Босс-файт, Адаптивное ДЗ — работают с любой темой |
| **Репетиторы** | Адаптивное ДЗ + Проверка Домашки + Босс-файт для мини-тестов |
| **Методисты** | Шаблоны геймифицированных уроков для распространения в школе |
| **Ученики** | Самопроверка конспектов, тренировка через босс-файты |
| **Родители (домашнее обучение)** | Квесты и игровые механики для мотивации ребёнка |
| **Вожатые / педагоги доп. образования** | Арена и Ход Игры для групповых мероприятий |
